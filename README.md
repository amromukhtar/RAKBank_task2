## Tech Stack

**Server:**  Axios Mock Adapter

## Features

- List Users
- Redux Toolkit Implementation

  
## Run Locally

Clone the project

```bash
  git clone https://github.com/amromukhtar/RAKBank_task2.git
```


Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn start
```
